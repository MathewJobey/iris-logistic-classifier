## 🧠 ML Details

- **Model**: Logistic Regression  
- **Dataset**: Iris  
- **Features**:
  - Sepal Length & Width
  - Petal Length & Width  
- **Target**: Predicts one of `Setosa`, `Versicolor`, or `Virginica`
